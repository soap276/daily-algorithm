## 모험가 길드
``` python
N = int(input())
data = list(map(int, input().split()))

# N = 5
# data = [2, 3, 1, 2, 2]

data.sort()
count = 0
i = 0
while N - data[i] > 0:
    count += 1
    N -= data[i]
    i += data[i]
print(count)
```
